**PYTHON GAME:**
A "Space Invaders"-style 2d shooter made from scratch, using Pygame. 

**INDEX**:

**Current_Iteration directory:** contains code, media files and documentation for latest version of the game.

**Final_Classwork_Version:** Final version of the game before submission as capstone for class (at which point it had 
	already far exceeded the class requirements)

**Graphic_Design directory:** examples of the aesthetic and creative work that went into making the game. More to come.

**Old_Code directory:** older iterations of the game, showing gradual building up of the code with manual additions as 
	concepts were learned. Built in VSCode, before began using Cursor and Git for version control.
