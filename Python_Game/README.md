PYTHON GAME:
A "Space Invaders"-style 2d shooter made from scratch, using Pygame. 

INDEX:

Current_Iteration directory: contains code, media files and documentation for latest version of the game.

Graphic_Design directory: examples of the aesthetic and creative work that went into making the game

Old_Code directory: older iterations of the game. Built in VSCode, before began using Cursor and Git for version control.
